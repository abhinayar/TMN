TrinitySDN virtualizes all of a device's connections and aggregates connection-states across the network. This provides network operators with a unified view of the entire network topology, including across heterogeneous networks such as cellular and Wi-Fi. The central server analyzes available connections.

Maintains an Always Best Connected state for each device in the network. Network operators can choose to configure their network for maximum throughput, maximum coverage, lowest operating cost.
