Simple Over-The-Top Deployment

Jumpnet was built with ease of deployment in mind. Jumpnet is a software-only solution that is 100% OTT. The Jumpnet SDK can be installed on all iOS 9.0+, Android 5.0+, and most other networked devices. Jumpnet’s server-side software does not require any integration with an LTE core or MSO regional data center to operate and integrate heterogenous networks. Unlike other attempts to integrate WiFi and cellular networks, Jumpnet does not require any changes to WiFi hotspots to provide link aggregation and seamless handoffs. Simply host the Jumpnet cloud installation in your data center and place the SDK into your existing mobile apps. That’s it!

Runs On Industry Standard Hardware

Jumpnet runs on industry standard hardware inside a cloud data center. It can even run on 3rd party hosted cloud deployments such as AWS or Rackspace inside your company’s managed infrastructure. Once installed, the Jumpnet server programs automatically connect and organize to set up the deployment. A web portal, hosted on your internal network, guides you through the process of setting up Jumpnet’s features inside your infrastructure.

The Jumpnet platform is designed for reliability and scale. The Jumpnet servers operate redundantly and failover services gracefully in the event of host failure. Depending on the capabilities of your deployment and feature requirements, Jumpnet is designed to easily reallocate network resources in real time, just like a true software defined network.

Link Aggregation

Jumpnet can provide faster, more reliable, and less expensive connections by combining available connections into a single virtual connection. With Jumpnet’s link aggregation, network operators can confidently offload more traffic to WiFi without compromising their subscribers’ QoE.

Seamless Handoffs

Jumpnet seamlessly integrates WiFi and cellular networks, so subscribers can move between networks without experiencing an interruption in connectivity.

Unified Network Monitoring

Jumpnet’s unified network monitoring shows subscribers’ entire connection experience on both primary and 3rd party networks. Network operators can see their subscribers connection experience from the perspective of endpoints to better plan and manage networks

Software Defined Networking +
Network Functions Virtualization

Jumpnet is build using software defined networking (SDN) and network function virtualization (NFV) principals. When we designed Jumpnet, we broke the key components of mobile, heterogeneous networks down to the functional units. After years of research into the best SDN and NFV practices and prototypes of the key elements of networks, we built Jumpnet from the ground up to have control and monitoring capabilities at all layers of the networking stack.

At its core, Jumpnet provides a centrally managed controller that can be distributed throughout the network, making for a responsive, robust, and real-time network engine. Jumpnet’s supports a wide range of network protocols, radios, and standards, and provides a unified monitoring and storage API for information generated by the network.

With Jumpnet, you can view the state of all connected nodes in the network in real-time, reconfigure the number of nodes, and get real-time notifications of relevant network events. What’s more, Jumpnet endpoints become part of the network infrastructure. Whether in the data center, in a smartphone, a router, or IoT device, the Jumpnet SDK can be used in any of these installations to build customized SDN applications that monitor relevant events and control other Jumpnet-enabled devices. Jumpnet is an over-the-top management layer for unifying networks and devices together.
